#  Sintaxe LeetCode

Repositório focado no estudo prático de sintaxes de linguagens de programação com base em:
- Fixação de sintaxe
- Lógica de programação
- Exercícios progressivos estilo LeetCode

---

## Estrutura

Cada pasta representa um **tema da linguagem**:

- streams-java
-
-

Dentro de cada tema:

- `README.md` → explicação + desafios
- `Ex##_Nome/` → exercícios resolvidos

---

## 🧠 Método de Estudo

Para cada tema:

1. Aula rápida (conceito direto)
2. Exercícios:
    - Easy → sintaxe
    - Medium → lógica
    - Hard → refatoração
3. Revisão com SRS (Anki)

---

## 📌 Convenção de Nomes

- Exercícios: `Ex01_Nome`
- Classes: `Main.java` ou nome do problema
- Código simples e focado
